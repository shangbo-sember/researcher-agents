# 中国低空经济市场规模数据（2023-2025）

## 核心数据摘要

### 市场规模增长
| 年份 | 市场规模 | 同比增长 | 数据来源 |
|------|----------|----------|----------|
| 2022年 | ~4000亿元 | - | 艾媒咨询 |
| 2023年 | 5059.5亿元 | 33.8% | 赛迪顾问/中商产业研究院 |
| 2024年 | 6700-9700亿元 | ~35% | 多方预测 |
| 2025年预测 | 1.5万亿元 | ~50% | 中商产业研究院/民航局 |
| 2030年预测 | 2-3.5万亿元 | - | 民航局/艾媒咨询 |
| 2035年预测 | 3.5万亿元 | - | 艾媒咨询 |

### 关键里程碑
- **2023年**：低空经济规模突破5000亿元
- **2024年**：首次写入政府工作报告，成为"新增长引擎"
- **2025年**：预计突破万亿规模，成为万亿级产业
- **增速**：2023年增速33.8%，预计2024-2025年保持30%+高增长

### 数据来源对比
| 来源 | 2023年 | 2024年预测 | 2025年预测 | 可靠性 |
|------|--------|-----------|-----------|--------|
| 赛迪顾问 | 5059.5亿元 | - | - | 高 |
| 中商产业研究院 | 5059.5亿元 | 9702.5亿元 | 1.5万亿元 | 中高 |
| 艾媒咨询 | - | 4807亿元 | - | 中 |
| 民航局 | - | - | 1.5万亿元 | 高 |
| 新华网/经济参考报 | 5059.5亿元 | - | 1.5万亿(2025)/3.5万亿(2035) | 高 |

**注**：不同机构统计口径略有差异，赛迪顾问和民航局数据最为权威

---
*数据来源：赛迪顾问《中国低空经济发展研究报告（2024）》、中商产业研究院、艾媒咨询、中国民航局、新华网*
